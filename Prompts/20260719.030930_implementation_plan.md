# Load Game Levels from SD Card

This plan details how to load Unblock Me game levels dynamically from the SD card using JSON files instead of hardcoded levels.

## User Review Required

> [!WARNING]
> Because your app targets Android 14 (API 34), accessing arbitrary folders on the SD card (like `/sdcard/Vypeensoft/...`) requires the `MANAGE_EXTERNAL_STORAGE` permission due to Scoped Storage. I will add logic to request this permission when the app starts.

## Open Questions

> 3. How should colors be assigned to other cars? (I plan to assign random or hash-based colors, keeping `"X"` as Red).
